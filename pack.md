# Pack file specification
The main modpack file for a packwiz modpack

## Example
```toml
name = "Shattered Minecarts"

[index]
file = "index.toml"
hash-format = "sha256"
hash = "e3d71dc5366997d84cab2d0b748a79bd8c3ad3433557a49dfac6cf2c7292d011"

[versions]
forge = "14.23.5.2838"
minecraft = "1.12.2"
```

## Properties
### Name
The name of the modpack

**Type:** string
